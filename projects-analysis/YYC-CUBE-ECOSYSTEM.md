## 🤖 技术栈总览

### 前端技术栈
- **框架**: Next.js 14/15/16, React 18/19
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: React Context, Redux
- **拖拽**: React DnD

### 后端技术栈
- **架构**: 微服务架构
- **容器化**: Docker
- **API**: RESTful, GraphQL

- **内网穿透**: FRP
